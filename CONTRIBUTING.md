# Contributing to ARP

Thank you for your interest in contributing to the Agent Relations Protocol.

## How to contribute

- **Report issues**: Open a [GitHub issue](https://github.com/Clerkboard/arp/issues) for bugs, spec ambiguities, or feature requests.
- **Propose changes**: Submit a pull request. For spec changes, please open an issue first to discuss the proposal.
- **Reference implementations**: Contributions to the TypeScript, Python, and Cloudflare Workers implementations are welcome.

## W3C Community Contributor License Agreement

ARP is being submitted as a W3C Community Group specification. Once the Community Group is approved, all contributors to the specification text will need to sign the [W3C Community Contributor License Agreement (CLA)](https://www.w3.org/community/about/process/cla/).

The CLA ensures that contributions can be freely used by anyone.

## Code of conduct

This project follows the [W3C Code of Ethics and Professional Conduct (CEPC)](https://www.w3.org/policies/code-of-ethics-and-professional-conduct/).
